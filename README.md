# Verification Code for the Paper

This repository provides a test script (`NMM.py`)  
for verifying the experimental results reported in the paper submitted to  
*Computers & Electrical Engineering (CAEE)*.

Only the inference (testing) code is released for verification purposes.  
The full training code and dataset are not shared due to confidentiality restrictions.
